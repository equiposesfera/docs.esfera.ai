en este video aprenderás a exportar tu proyecto desde esfera ams project una herramienta clave para la planificación y gestión de proyectos para acceder a esta función puedes ir al menú obra y seleccionar presupuesto o simplemente hacer clic en el icono de presupuesto ubicado en la parte superior de cualquier pantalla dentro de esfera que te llevará amente a la ventana correspondiente la integración de esfera con MS project permite que toda la planificación de tu proyecto se refleje de manera precisa en esta herramienta el listado de ítems de tu presupuesto se convierte en tareas dentro de MS project respetando las cantidades computadas para cada etapa del proyecto Por ejemplo si un ítem está distribuido en cuatro etapas se exportará como cuatro tareas distintas en ms Project fielmente la planificación definida en esfera además esfera transfiere los recursos asociados a cada ítem y calcula la duración de las tareas según los rendimientos definidos en el análisis de precios unitarios de la mano de obra para facilitar este proceso primero debes hacer clic en el botón ver plantilla MS project lo que abrirá una ventana para descargar el manual de usuario y los archivos necesarios para implementar la cinta de opciones y la Macro personalizada de esfera en ms project esta cinta está diseñada exclusivamente para mejorar la experiencia de los usuarios Y contiene funcionalidades específicas que explicaremos más adelante una vez configurada la cinta de opciones siguiendo las instrucciones del manual regresa a la ventana de presupuesto en esfera y haz clic en el botón exportar ams project este proceso puede tomar unos minutos cuando finalice podrás abrir el aro de tu proyecto en ms project donde encontrarás la cinta de opciones de esfera en la parte superior los ítems de tu presupuesto aparecerán como tareas en la columna correspondiente y aquellos que pertenecen a varias etapas se exportarán Tantas veces como sea necesario la duración de las tareas se calcula a partir de los rendimientos de mano de obra establecidos en esfera desde la cinta de opciones de esfera podrás organizar las tareas según diferentes criterios como etapas categorías o grupos facilitando la visualización y el seguimiento del proyecto Por ejemplo puedes organizar las tareas por etapa lo que mostrará solo aquellas que pertenecen a cada etapa correspondiente también puedes ordenarlas por etapas y categorías o por grupos y etapas según las necesidades del proyecto a modo de ejemplo colocaré las predecesoras de forma correlativa en la sección de ruta crítica puedes visualizar las tareas críticas del proyecto organizándolas por etapas categorías o ambas las actividades con holgura cero se resaltarán en rojo permitiéndote identificar fácilmente los elementos críticos que afectan el plazo del proyecto la sección de agrupaciones financieras te permite analizar el flujo financiero de las tareas por etapa categoría o una combinación de ambas mostrará primeramente el flujo financiero de la mano de obra haciendo clic derecho sobre el panel podrás visualizar costos asociados utilizando las opciones avanzadas de MS project otras funcionalidades útiles incluyen la opción de planificación que te permite regresar al orden original de las actividades eida tarea que ubica el inicio de una actividad específica en el diagrama de gant también puedes establecer una línea base desde la cinta de opciones de esfera utilizando la opción plan previsto o ajustar fechas de corte para actividades seleccionadas finalmente puedes exportar toda la información a Excel para realizar análisis adicionales es importante destacar que el aprovechamiento de todas estas funcionalidades dependerá del conocimiento del usuario sobre MS project en el próximo video exploraremos cómo Dar seguimiento a la planificación cuando ya se hayan registrado avances y compras en esfera también profundizaremos en otras funcionalidades de la cinta de opciones de esfera diseñadas para optimizar la gestión durante la ejecución del proyecto
